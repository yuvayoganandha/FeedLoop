# 🍏 FeedLoop: Waste Food Donation Platform

> **Redistributing Surplus. Reducing Waste. Feeding Communities.**

**FeedLoop** is a streamlined web application designed to bridge the gap between food donors (individuals, restaurants, hotels) and those who can rescue and redistribute surplus food. Our goal is simple: ensure that no edible food goes to waste.

---

## ✨ Key Features

- **📍 Live Food Radar**: View a real-time map of available food donations in your local area.
- **📝 Easy Donation Form**: Quickly post details about surplus food, including quantity, type, and expiry time.
- **⚡ Real-time Sync**: Instant updates via WebSockets ensure claimed food is removed from the map immediately.
- **👤 User Dashboard**: Track your donation history and see the impact you've made in your community.
- **📱 Responsive Design**: Works seamlessly on mobile and desktop devices.

---

## 🛠️ Technical Overview

### Frontend
- **React.js**: Modern component-based architecture.
- **Tailwind CSS**: Sleek, responsive styling.
- **Leaflet.js**: Interactive map for real-time tracking.

### Backend
- **Node.js & Express**: Robust and scalable server.
- **MongoDB Atlas**: Cloud database for persistent data.
- **Socket.io**: Real-time communication for instant claims.

---

## 🚀 How to Run FeedLoop

### 1. Backend Server
```bash
cd backend
npm install
# Configure your .env (see .env.example)
node server.js
```

### 2. Frontend Application
```bash
cd frontend
npm install
npm run dev
```
